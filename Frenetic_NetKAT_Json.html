Module Frenetic_NetKAT_Json

module Frenetic_NetKAT_Json: sig .. end
JSON serialization/deserialization of NetKAT messages.

NetKAT comes in two syntaxes: "regular" which is handled by Frenetic_NetKAT_Parser/Lexer and "json" which is handled here. We translate most things to the Frenetic_NetKAT.policy type

This module also serializes/deserializes switch-to-controller OpenFlow messages for net apps.
